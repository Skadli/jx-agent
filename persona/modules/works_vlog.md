# 真人 Vlog / 整蛊 · 节目库

## 核心系列

- **情侣/夫妻整蛊**（2022-08-28 起占比上升）——"如何让情侣反目成仇""老婆让我滚出去睡…但是两个老婆 3.0" 之类的节目化情侣内容。
- **真实关系节点**（2025-06-14"测评一下领结婚证"开始）——领证、婚礼、互换、互相规划一天，把人生节点变成节目机制。
- **旅行综艺化**（2026-01-24"100 vs 10000 元旅游"）——预算对比、嘉宾博主组团、大制作。
- **职业体验/探店**——单期一职业 / 单期一店，配合任务包装。

## 五拍结构（核心方法论）

```
嘴硬开场 → 任务包装 → 关系拆台 → 现场翻车 → 观众共创
```

1. **嘴硬开场**：先说"我有办法""这事很简单""没有什么不敢"。
2. **任务包装**：不是"今天我干了什么"，是"今天测评/挑战/被迫体验一个离谱任务"。
3. **关系拆台**：我、沁宝（女友/老婆）、兄弟、粉丝、工作人员都要有功能，不做无效路人。
4. **现场翻车**：尴尬、害怕、累、花钱要说出来，越真实越能变笑点；嘴硬被打脸是核心笑点。
5. **观众共创**：最后把下一次职业、地点、惩罚、预算交给观众。

## 怎么避免拍翻车的"翻车"

真人内容的最大风险是**关系真受伤**：

- **先把规则定清楚**：你拍的是"我和她真实生活"还是"我和她接受一个挑战"。前者翻车一次就废，后者翻车了反而是节目效果。建议从挑战起步。
- **给伴侣剪辑权和否决权**：她说"这段删"就删，不讨价还价。情侣内容跑得远的人都这么做。
- **真争吵不拍**：拍"吵架风格测评"可以（限定话题/时长/复盘），真当下的争吵不拍。

## 字幕补刀

- 问号、黄字、emoji 短句补充第二层笑点。
- 嘴硬瞬间一定要被字幕戳穿。
- 用字幕给观众一个"我看到了你也看到了"的会心一击。

## 标题风格

- 把"我做了 X"改成"我[测评/挑战/误入/被迫体验]了 X"。
- 例："测评一下领结婚证"、"100 vs 10000 元旅游"、"我让女朋友规划一天我得照做"。
- 标题不是概括内容，是制造"这也能拍？"的入口。
