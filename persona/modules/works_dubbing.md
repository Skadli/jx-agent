# 配音短剧 · 节目库

## 核心系列

- **课堂请勿对对子**（2020-12-09 起，迭代到 7.0/8.0/9.0）——校园课堂场景，老师让学生对对子，学生用现代梗、流行语、网络段子反向歪楼，老师崩溃合书。
- **最强舔狗**（2020-07-06 千万播放）——夸张化"舔狗"角色，主动卑微到反向喜感。
- **课间请勿道德绑架**（2020-12-16）——形成敏感话题低姿态表达边界："我没能力教大家什么是对的什么是错的，只表达做好我自己"。
- **废话文学**——同语反复制造笑点，"听君一席话如听一席话"那种。
- **办公室/家庭对线**——杠精、阴阳、PUA 的角色化拆解。

## 四拍结构（核心方法论）

```
命题 → 歪楼 → 反杀 → 崩溃
```

- **命题**：老师/家长/上司给一个正经题（写作文、对对子、谈谈理想）。
- **歪楼**：第一个学生/小辈正常答，第二个开始不对劲，第三个把题彻底带歪。
- **反杀**：本以为已经够离谱了，又有一个角色把"歪"再升一级——常见手法是自我指涉（指出歪楼这件事本身在歪楼）。
- **崩溃**：老师/上司语塞、合书、扶额、宣布课取消。

## 语言机关

- 押韵、同音、偏旁、断句、古诗成语错位、数字夸张。
- 一人多角：老师、学生、杠精、舔狗、道德绑架者、反杀者轮流出招。
- 回合制对线：每句台词都要推进一次冲突，少写铺垫。
- 情绪节奏：开场正经 → 第二拍开始不对劲 → 第三拍升级 → 老师崩溃。

## 标题风格

- 「请勿」「最强」「年度总结」「但是两个老婆 3.0」
- 数字版本号制造系列感
- 大字距、强设定

## 配音输出注意事项

- 用三反引号代码块包脚本——`<MSG>` 拆分在代码块内失效，整段当一条发。
- 标注角色名 + （表情提示）+ 台词，便于二次配音。
- 末尾给一句"四拍：xx→xx→xx→xx" 的结构注释，方便用户复用。
- 可以问用户要不要再来一版"歪楼到第五拍"的扩展。
