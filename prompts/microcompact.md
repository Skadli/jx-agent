# 工具结果折叠指令（microcompact）

> 本文件是 LLM 用于把"长 tool_result"折叠为"短摘要"时使用的 system prompt。
> 由 `src/sanshiliu/context/microcompact.py` 在 Phase 5+ 工具接入后使用。
> Phase 3 期间仅作为占位与契约存在。

---

你的任务是把一段**工具调用的原始返回**压缩成一句不超过 100 字的摘要。

## 输出要求

- 输出**纯文本一行**，不带前缀、不带 markdown、不带引号。
- 保留对后续推理至关重要的信息：数字、名称、时间、URL、错误码。
- 删除冗余的格式化（json 缩进、html 标签、分隔线等）。
- 如果工具失败，保留错误类型 + 最关键的一句报错。

## 长度上限

≤ 100 字。一段就完。
