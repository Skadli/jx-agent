# Teochew Conversation Guide

Use this file when composing Teochew replies, translating into Teochew, or checking whether the output is Teochew rather than Mandarin.

## Core Vocabulary

| Function | Common form | Peng'im | Notes |
| --- | --- | --- | --- |
| I | 我 | ua2 | |
| you | 汝 | le2 / lu2 | Region and spelling vary. |
| he/she | 伊 | i1 | |
| we, inclusive | 俺 | nang2 | Includes the listener. |
| we, exclusive | 阮 | uang2 / ung2 | Excludes the listener. |
| you all | 恁 | ning2 | |
| be | 是 | si6 | |
| not be | 唔是 | m6-si6 | |
| have | 有 | u6 | |
| no / not have | 無 | bho5 | |
| want | 愛 | ai3 | Also appears in want-or-not questions. |
| eat | 食 | ziah8 / jiah8 | Regional romanization varies. |
| possessive / modifier | 個 | gai5 | `汝個` = yours/your. |

## Negatives

Teochew negatives are not one-size-fits-all:

- `唔` m6: prefix for many basic negations, e.g. `唔是`.
- `無` bho5: no, not have, not done in many contexts.
- `袂` bhoi6: cannot / unable / not capable.
- `勿` mai3: not want / do not, often paired with `愛` ai3.
- `免` mêng2 / miêng2: no need; gentler refusal.
- `未` bhuê7: not yet.

Pick the semantic negative, not the Mandarin-looking one.

## Questions

- Yes/no particle: add `me7` at the end when the sentence itself is an affirmative statement.
- Skeptical question: use `me5`.
- Confirmation: use `han2`, roughly "right?".
- Clause-negative: add `a1bho5` for "or not".
- Want-or-not: `愛 ... 呀勿` / `ai3 ... a1mai3`.
- Not yet: use `未` / `bhuê7`.
- Reciprocal or softened question: add `nê5` when appropriate.

## Sentence-Final Particles

Use particles to mark stance:

| Particle | Use |
| --- | --- |
| `nê5` | reciprocal question, "what about...?", or softened question |
| `li1` | softening or affirming something questioned |
| `ho2` | seek acknowledgement or confirmation |
| `no7` | emphasis or realization |
| `ma1` | certainty |
| `a1` | emphasis, often in questions or clause-negative patterns |

Do not use particles as decoration. If unsure, leave the sentence plain.

## Output Strategies

For normal chat:

```text
我先照常見潮汕話寫；若你愛汕頭、潮州、揭陽某一地口音，我再幫你調。
```

For learning:

```text
汝愛食茶呀勿？
Le2 ai3(2)ziah8 de5 a1mai3?
你要不要喝茶？
```

For uncertain characters:

```text
Characters vary for this word; I will give Peng'im first and then a readable character version.
```

## Anti-Patterns

- Do not write Mandarin sentences and swap `你` for `汝`.
- Do not use Cantonese words like `佢`, `咗`, `喺`, `嘅` unless explicitly comparing languages.
- Do not imply that all Chaoshan regions pronounce the same sentence identically.
- Do not overuse rare characters when romanization is clearer.
