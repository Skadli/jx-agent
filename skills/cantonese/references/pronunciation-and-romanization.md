# Cantonese Pronunciation And Jyutping

Use this file when the user asks for pronunciation, romanization, tones, reading aloud, or language-learning help.

## Jyutping Basics

Use Jyutping as the default romanization. Write syllables in lowercase and append tone numbers: `nei5 hou2`, `gwong2 dung1 waa2`.

The six standard Cantonese tone numbers are:

| Tone | Contour | Typical label |
| --- | --- | --- |
| 1 | high level or high falling | 陰平 |
| 2 | high rising | 陰上 |
| 3 | mid level | 陰去 |
| 4 | low falling | 陽平 |
| 5 | low rising | 陽上 |
| 6 | low level | 陽去 |

Entering-tone syllables ending in `p`, `t`, or `k` use tone values that map to tone numbers 1, 3, or 6 in Jyutping practice: `sik6`, `faat3`, `jat1`.

## Output Formats

- Simple pronunciation: `多謝你。do1 ze6 nei5.`
- Learning format: `多謝你 (do1 ze6 nei5) = thank you.`
- Tone-focused format: list each word with Jyutping: `我 ngo5 / 今日 gam1 jat6 / 有 jau5 / 時間 si4 gaan3`.
- Do not use Hanyu Pinyin for Cantonese.

## Accuracy Notes

- Keep `j`/`z`/`c` distinctions in Jyutping: `zi2`, `ci4`, `jyut6`.
- Use `yu` for the front rounded vowel: `jyut6`, `syu1`, `heoi3`.
- Use `ng` as an initial where standard Jyutping has it: `ngo5`, `ng5`.
- Do not mark every casual chat reply with Jyutping; it makes normal conversation hard to read.
