# Cantonese Conversation Guide

Use this file when composing Cantonese replies, translating into Cantonese, or checking whether output sounds like natural Cantonese.

## Core Vocabulary

| Function | Cantonese | Jyutping | Notes |
| --- | --- | --- | --- |
| I / you / he/she | 我 / 你 / 佢 | ngo5 / nei5 / keoi5 | Use `佢` for people and sometimes personified things. |
| we / you all / they | 我哋 / 你哋 / 佢哋 | ngo5 dei6 / nei5 dei6 / keoi5 dei6 | Do not use Mandarin-style `我們`. |
| be | 係 | hai6 | `我係...` |
| not be | 唔係 | m4 hai6 | `呢個唔係問題。` |
| have / not have | 有 / 冇 | jau5 / mou5 | `冇` also covers "did not" in many contexts. |
| at / in | 喺 | hai2 | `我喺屋企。` |
| possessive / modifier | 嘅 | ge3 | `你嘅諗法`, `好用嘅方法`. |
| some / plural-ish / small amount | 啲 | di1 | `啲資料`, `順啲`. |
| this / that | 呢 / 嗰 | ni1 / go2 | `呢個`, `嗰邊`. |
| like this | 咁 | gam2 | `咁樣`, `咁講`. |

## Grammar Patterns

### Aspect

- Completion: `V + 咗`: `我睇咗。`
- Progressive: `V + 緊`: `我而家寫緊。`
- Experience: `V + 過`: `我試過呢個方法。`
- Not yet: `未 + V`: `我未睇。`
- Did not / no: `冇 + V`: `我冇改到。`

### Questions

- Yes/no: `係咪...？`, `...係咪？`
- Have / did: `有冇...？`
- Not yet: `...未？`
- Ability / permission: `可唔可以...？`, `得唔得？`
- What / why / when: `咩`, `點解`, `幾時`, `邊個`, `邊度`, `點樣`.

### Natural Rewrites

Avoid Mandarin-shaped Cantonese:

- `我正在看这个文件` -> `我睇緊呢份文件。`
- `这个问题已经解决了` -> `呢個問題已經解決咗。`
- `你有没有时间？` -> `你有冇時間？`
- `我不知道他们在哪里` -> `我唔知佢哋喺邊。`
- `我们可以这样做` -> `我哋可以咁做。`

## Register

- Casual friend: `你想我幫你點搞呀？`
- Polite assistant: `我可以幫你整理成粵語版。`
- Professional but colloquial: `呢個寫法會自然啲，亦都唔會太口語。`
- Formal writing: switch to Standard Written Chinese if the user asks for an official notice, legal text, or academic prose.

## Particle Use

Particles change stance, not just flavor. Use one or two per sentence at most.

| Particle | Use |
| --- | --- |
| 呀 | soften a prompt or question |
| 啦 | suggest, conclude, or mark a changed state |
| 喎 | report new or contrastive info |
| 咩 | skeptical/surprised question |
| 㗎 | explain or assert a fact |
| 囉 | obvious/resigned conclusion |
| 啫 | "just/only", often softening |

## Anti-Patterns

- Do not translate Mandarin particles mechanically: `了` is not always `咗`; sometimes it is `啦`, sometimes no marker.
- Do not overuse `啦呀喎㗎咩` in every sentence.
- Do not write "Cantonese" by keeping Mandarin word order and replacing a few words.
- Do not use Simplified Chinese unless requested; Hong Kong-style Cantonese normally uses Traditional Chinese.
