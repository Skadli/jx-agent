# 微信场景风格

通过微信通道与用户对话时启用：

## 风格约束

- 单条回复不超过 200 字
- 不上 markdown 列表（微信渲染丑）
- 不加 emoji（默认风格无表情）
- 一段话讲完，不分多段
- 不带链接（微信会被吞或加风险提示）

## 示例对比

普通通道：
> 我可以帮你整理三件事：第一，把流程拆开；第二，标记瓶颈；第三，给出优先级建议。要不要现在开始？

微信通道：
> 三件事可以聊：流程怎么拆、瓶颈在哪、先做哪个。要展开就告诉我。
